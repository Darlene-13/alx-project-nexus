🧠 Machine Learning Examples

🎯 Get Personalized Recommendations:

GET /recommendations/v1/recommendations/personalized/?limit=10&refresh=true Authorization: Bearer <token>

🧪 A/B Test Algorithm Performance:

POST /recommendations/v1/experiments/ { "name": "Algorithm Comparison", "algorithm_a": "collaborative_filtering", "algorithm_b": "deep_learning", "traffic_split": 0.5 }

📊 Track User Interactions:

POST /recommendations/v1/interactions/ { "movie_id": 123, "interaction_type": "like", "rating": 8.5, "feedback_type": "positive" }

🎭 Complete User Onboarding:

POST /recommendations/v1/users/onboarding/ { "favorite_genres": [{"genre_id": 28, "preference_score": 0.9}], "content_rating_preference": "PG-13", "diversity_preference": 0.7 }

🔬 ML Algorithm Insights

🧠 Available Algorithms:

  • Collaborative Filtering: Finds users with similar preferences
  • Content-Based: Recommends based on movie attributes
  • Deep Learning: Neural networks for complex patterns
  • Hybrid Approach: Combines multiple algorithms

🎯 Personalization Features:

  • Real-time user preference learning
  • Dynamic algorithm selection based on user behavior
  • Cold start handling for new users
  • Diversity and novelty controls
  • A/B testing for algorithm optimization
