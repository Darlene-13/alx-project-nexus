💡 Usage Examples

🔍 Search Movies:

GET /movies/api/v1/search/?q=batman&genre=Action&year=2020&rating_min=7.0

🎯 Get Recommendations:

GET /movies/api/v1/recommendations/?type=popular&limit=10

📊 Movie Analytics:

GET /movies/api/v1/analytics/

🎬 Get Popular Movies:

GET /movies/api/v1/movies/popular/?limit=20

📋 Quick Reference

🎬 Key Features:

  • Complete CRUD operations for movies and genres
  • Advanced search with multiple filters
  • AI-powered recommendations
  • Real-time analytics and statistics
  • Genre-based categorization
  • View and like tracking

🔍 Search Parameters:

  • q - Search query (title, overview, director)
  • genre - Filter by genre name
  • year - Filter by release year
  • rating_min/max - Filter by TMDB rating range
  • limit - Number of results to return

Base URL: /movies/

Authentication: Token-based (optional for read-only)

Response Format: JSON with pagination
